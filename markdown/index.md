# OpenSecretary

An autonomous AI agent for Obsidian.

Read, write, search, and organize your notes with an intelligent assistant. Works on desktop and mobile.

## Install with BRAT

1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat) from Obsidian Community Plugins
2. Open BRAT settings and select "Add Beta Plugin"
3. Paste the repository URL: `https://github.com/shanemtosh/open-secretary`
4. Enable OpenSecretary in Community Plugins
5. Add your [OpenRouter](https://openrouter.ai) API key in settings

## Features

### Desktop + Mobile

Works everywhere Obsidian runs. Unlike traditional coding agents, use it on your phone.

### Open Source

AGPLv3 licensed. Free to use, modify, and share. Community-driven development.

### Bring Your Own Key

Use your OpenRouter API key. Choose from dozens of models. You control the costs.

### File Operations

Read, write, edit, delete, move, and organize files autonomously in your vault.

### Sub-Agents

Specialized agents for research, writing, and vault exploration.
